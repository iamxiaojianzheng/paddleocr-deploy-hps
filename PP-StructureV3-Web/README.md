# PP-StructureV3 Web 测试预览工具

本项目是针对 **PP-StructureV3** 文档解析服务开发的轻量级 Web 客户端。它提供了一个直观的交互界面，支持多种配置参数，以辅助测试和预览服务的推理效果。

## 功能特性

1. **多参数配置**：
   - 支持常规功能开关：印章识别、表格识别、公式识别、复杂版面处理、版面方向矫正、扭曲图像矫正等。
   - 支持高级参数微调：版面检测框阈值、NMS后处理、文本检测/识别各种限制等。
2. **多模态结果预览**：
   - **Markdown 预览**：将接口返回的文本渲染为排版优美的文档，并将返回的段内切图以 Base64 嵌入图片节点直接进行内联展示。
   - **可视化图**：展示版面检测模型对各个块（文本行、公式、表格、印章、图像等）的划分定位。
   - **输入对比**：直观展示上传的原图或 PDF 页。
   - **JSON 高亮树**：提供支持节点折叠与展开、一键复制的高亮 JSON。
3. **高颜值设计**：
   - 玻璃拟态 (Glassmorphism) 毛玻璃质感面板。
   - 自适应 Light / Dark 浅色和暗色主题一键切换。
4. **动态多页分页**：对上传的多页 PDF，将结果分段分页展示，方便独立翻页预览。
5. **CORS 本地代理**：
   - 内置极简的 python 跨域代理中转服务（`cors_proxy.py`），无缝打通本地静态网页与 Docker 中的 API 服务。

## 项目结构

```text
PP-StructureV3-Web/
├── css/
│   ├── main.css          # 全局重置、主题变量
│   └── components.css    # 各种 UI 组件样式 (如开关、表单、高亮 JSON)
├── js/
│   ├── api.js            # fetch 网络请求封装及 Base64 转换
│   ├── params.js         # 表单参数收集与 JSON 数据解析
│   ├── results.js        # Markdown 渲染替换、图片预览及 JSON 树解析
│   └── app.js            # 事件监听初始化与主题切换逻辑
├── index.html            # 主双栏界面
├── cors_proxy.py         # 跨域中转代理脚本 (8082 -> 8081)
└── README.md             # 本说明文件
```

## 快速开始

### 第一步：启动 Triton & Gateway Docker 服务
确保您的 PP-StructureV3 产线已在 Docker 中成功跑起，并且 gateway 网关已暴露并运行在本地的 `8081` 端口。

### 第二步：启动跨域中转代理 (CORS Proxy)
由于浏览器存在跨域限制，在终端运行以下 Python 脚本，以启动本地代理：

```bash
python cors_proxy.py
```

终端将打印：
```text
============================================================
 PP-StructureV3 CORS 跨域代理服务器已启动!
 代理服务监听地址: http://localhost:8082
 请求将自动转发至 Docker 服务: http://localhost:8081
 请在前端页面的 'API Base URL' 中填写: http://localhost:8082
============================================================
```

### 第三步：打开前端界面
直接双击打开 `index.html` 即可开始使用，或者使用您的任意本地静态服务器（如 `Live Server` / `http-server`）托管当前目录。
在网页顶部的 **API Base URL** 中输入 `http://localhost:8082`。然后上传您的图像或 PDF 文档并点击 **发送 API 请求**。
